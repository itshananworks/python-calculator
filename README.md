# Python Calculator

A simple calculator application built with Python.

## Features

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Input validation
* User-friendly menu interface
* Continuous calculation loop until user exits

## Technologies Used

* Python 3

## How to Run

1. Clone this repository

```bash
git clone https://github.com/itshananworks/python-calculator.git
```

2. Open the project folder

```bash
cd python-calculator
```

3. Run the program

```bash
python calculator.py
```
